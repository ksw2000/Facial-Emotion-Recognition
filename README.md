# Facial emotion recognition

A simple APP can recognize facial emotion by camera.

They are two modes.

1. Non real time
2. real time



## Getting Started

This is project is developed on Android now.

1. Get dependencies

   ```shell
   flutter pub get
   ```

   

2. To initialize `tflite_flutter` package, execute `install.bat` before running.

   ```shell
   ./install.bat
   ```

3. Running

   ```shell
   flutter run
   ```

4. or Build apk

   ```shell
   flutter build apk
   ```
